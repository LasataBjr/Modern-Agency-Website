# Modern Creative Agency Website

A modern and responsive creative agency landing page built using HTML, CSS, JavaScript, and jQuery.  
This project focuses on clean UI/UX design, responsive layouts, smooth interactions, and modern web aesthetics.

---

## Project Overview

This project was created to practice and improve frontend development skills while following a professional UI/UX workflow.

The website is designed as a modern digital agency landing page featuring:
- Responsive layouts
- Smooth scrolling
- Interactive sections
- Modern typography
- Professional UI design
- Mobile-friendly navigation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Figma

---

## Features

- Responsive design for all devices
- Sticky navigation bar
- Mobile hamburger menu
- Smooth scrolling navigation
- Animated hover effects
- FAQ accordion
- Testimonial section
- Modern card-based layout
- Clean UI/UX structure
- Optimized section spacing

The project development process included both design and frontend implementation.

### Design Workflow

- Semantic HTML structure

- Responsive layout implementation
